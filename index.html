Qus-1: how does javascript works?

Ans: JavaScript is an interpreted language, not a compiled language. javascript is programming language and client-side scripting language. Browser scripting remains the main use-case of JavaScript. It totally depends on the web-developers and it can be used for several things in web development.The language was created to allow web developers to embed executable code on their webpages, so that they could make their webpages interactive.

Qus-2: local storage and session storage difference?

Ans: Local storage with this if i store my data, it will be permanent, I will close the browser the data will remain. If we take any data from the user, we can keep it permanebtly in the loval storage. Session storage will only have one session. If you close the browser, there will be no data. The data stored in localStorage and sessionStorage can easily be read or changed from within the browser so should not be relied upon for storage of sensitive or security-related data within applications
